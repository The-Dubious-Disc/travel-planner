# Travel Planner

**Overview:**
A web app to efficiently organize multi-city trips using an intuitive drag-and-drop interface with inputs for managing city order, duration, and visualizing the itinerary as a Gantt chart.

**Purpose:**
This tool helps users simplify complex trip arrangements by providing easy-to-use features such as summary views, constraints warnings for days and dates, and both desktop and mobile-responsive functionality.
